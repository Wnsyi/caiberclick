# Component 3 恋爱叙事双卡片 · 游戏设计

## 概述

Component 3 下方的两个小组件（comp3-card-a / comp3-card-b）各承载一个恋爱分支叙事游戏。点击卡片后跳转到 Chat 页，以"恋与深空"式的人物+对话框模式展开互动故事。

复用现有游戏引擎：`selectCard()` → `startPhase()` → `playMessages()` → 用户选择 → `finishGame()` → `renderResult()`。

---

## 数据结构

在 `EXPERIENCE_CARDS` 数组末尾新增 2 张卡，每张卡包含：

```
{
  id, emoji, badge, stars, reviews, title, desc,
  slideClass, imgSrc, avatarColors,
  startPhase, results: null,
  phases: { phase1, phase2a, phase2b, phase3a, phase3b, end }
}
```

## 与现有系统的集成

### HTML 改动
`comp3-card-a` / `comp3-card-b` 各加 `onclick="selectCard('love_crush')"` / `onclick="selectCard('love_reunion')"`

### JS 改动
1. `EXPERIENCE_CARDS` 数组新增 2 张卡（数据 + phases + 结果映射）
2. `buildResults()` 新增两组调用
3. 其余游戏引擎不动

### CSS 改动
新增 2 个颜色变体：
- `.slide-love-crush` — 玫瑰粉渐变
- `.slide-love-reunion` — 暮紫渐变
- `.comp3-card` 新增 `cursor:pointer` + hover 效果
